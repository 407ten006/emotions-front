# 🧑‍💻 비사이드 프론트엔드 프로젝트

> 사용자 친화적인 UI와 깔끔한 아키텍처를 기반으로 한 비사이드 프론트엔드 개발 프로젝트입니다.  
> 상태 관리부터 라우팅, 스타일링까지 최신 React 생태계 기반으로 구축하였습니다.

---

## 🚀 기술 스택

| 역할            | 사용 라이브러리       |
|-----------------|------------------------|
| 패키지 매니저   | [`Yarn Classic`](https://classic.yarnpkg.com/lang/en/) |
| 라우팅          | [`React Router`](https://reactrouter.com/) |
| 스타일링        | [`styled-components`](https://styled-components.com/) |
| 상태 관리       | [`Zustand`](https://github.com/pmndrs/zustand) |
| 폼 처리         | [`React Hook Form`](https://react-hook-form.com/) |

## 🧩 주요 기능

1. 🔐 회원 인증 기능
회원가입 / 로그인 / 로그아웃

사용자는 이메일과 비밀번호를 입력해 가입 가능

JWT 또는 세션 기반 인증으로 보호

비밀번호 찾기 / 초기화 기능 가능성

2. 🗃️ 게시판 기능
글 작성 / 조회 / 수정 / 삭제 (CRUD)

제목, 내용 입력 가능

사용자가 작성한 글은 본인만 수정/삭제 가능

글 목록 페이지

글 제목, 작성자, 등록일 등을 리스트로 확인 가능

페이징 처리 가능성 있음 (페이지 넘버 UI 존재)

3. 📝 댓글 기능
각 게시글에 댓글 작성 가능

작성된 댓글은 수정 / 삭제 가능

대댓글 기능은 이미지상에서는 안 보이지만, 확장 가능성 있음

4. 🔍 검색 및 필터링 기능
게시글 검색 기능

제목 기반 또는 내용 기반 검색 기능 존재

카테고리 필터링 (이미지 우측 상단 탭 기반 추정)

5. 👤 마이페이지 기능 (추정)
내가 쓴 글 / 댓글 조회

회원정보 수정 또는 탈퇴 기능 가능성 있음

6. 📱 반응형 UI 고려
전체 UI가 정갈하고 모바일 환경도 고려된 레이아웃
(글 목록과 작성 버튼의 위치, 폰트 크기 등에서 추정 가능)
